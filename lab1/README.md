##  Лабораторная работа №1

### Вариант 24(Модель умного дома)

Предметная область: автоматизация и управление домашней средой.

Важные сущности: умный дом, умные устройства, системы безопасности, управление освещением, климат-контроль, умные бытовые приборы.

Операции: операция управления освещением и климатом, операция мониторинга систем безопасности, операция управления умными устройствами через мобильное приложение, операция создания расписания автоматизации, операция дистанционного контроля за домом.
##  О программе
Программа представляет собой имитационную модель робота, оснащённого сенсорами, механическими манипуляторами, системой передвижения и программным обеспечением.

###Система способна:
Управлять освещением (включение, выключение, регулировка яркости)

Управлять климат-контролем (установка температуры)

Управлять пылесосом (обычная и влажная уборка)

Управлять чайником (кипячение, нагрев воды)

Обеспечивать безопасность (сигнализация с режимами охраны)

Создавать правила автоматизации по времени или статусу устройств

## 🎮 Интерфейс командной строки
При запуске программы отображается главное меню. Все команды вводятся цифрами.
