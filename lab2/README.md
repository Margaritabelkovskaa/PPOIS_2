# Система управления футболистами

## Описание проекта
Десктопное приложение для учета футболистов с графическим интерфейсом. Позволяет добавлять, искать, удалять записи, а также импортировать/экспортировать данные в XML.

## Функциональные возможности

### Основные операции
- **Добавление** новых футболистов через диалоговое окно
- **Поиск** записей по различным критериям с постраничным выводом
- **Удаление** записей по условиям с подтверждением
- **Просмотр** всех записей в главном окне
- **Альтернативное отображение** в виде дерева (группировка по командам)

### Работа с данными
- Хранение в **SQLite** базе данных
- Импорт/экспорт в **XML** (DOM для записи, SAX для чтения)
- 60+ тестовых записей для демонстрации

### Интерфейс
- Главное окно с таблицей записей
- Меню и панель инструментов (дублирование команд)
- Диалоговые окна для добавления, поиска, удаления
- **Постраничный вывод** с навигацией (первая, последняя, следующая, предыдущая)
- Календарь для выбора даты рождения

## Технологии
- **Python 3.13**
- **PyQt5** - графический интерфейс
- **SQLite** - база данных
- **XML** (SAX/DOM парсеры) - обмен данными
